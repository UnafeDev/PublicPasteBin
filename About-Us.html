Why do we collect login info?


Collecting login info allows users to add their own pastes and delete them, creating a login will only permit that user to remove their paste.


How is it stored?

  1. We do not store any passwords, we use Firebase Authentication to handle all login info.
  2. The only stored info is the user's email.
  3. Emails will never be shared publicly.

Is there a way to remove our account?

Yes, you can remove your account by going to the Account Deletion page and submitting a request.


Why make this if Pastebin exists?

This project was made for educational purposes, to learn how to use Firebase and make a simple web app. It is not intended to compete with Pastebin or any other similar service. Simply a high-school project
